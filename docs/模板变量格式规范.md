# 模板变量格式规范

## 概述

lcyf-claude-code v2.0 使用 **Mustache** 语法作为模板变量格式，简单、强大、易于理解。

## 基本语法

### 1. 简单变量替换

```java
// 模板
package {{packageName}};

public class {{className}} {
    private {{fieldType}} {{fieldName}};
}

// 使用
packageName: "com.lcyf.cloud.module.sales"
className: "UserService"
fieldType: "String"
fieldName: "userName"

// 结果
package com.lcyf.cloud.module.sales;

public class UserService {
    private String userName;
}
```

### 2. 条件渲染

```java
// 模板
public class {{className}} {
    {{#hasCache}}
    @Cacheable(value = "{{cacheName}}")
    {{/hasCache}}
    public {{returnType}} get{{entityName}}(Long id) {
        return {{entityName|lowercase}}Mapper.selectById(id);
    }
}

// 使用
hasCache: true
cacheName: "user"
returnType: "User"
entityName: "User"

// 结果
public class UserService {
    @Cacheable(value = "user")
    public User getUser(Long id) {
        return userMapper.selectById(id);
    }
}
```

### 3. 列表渲染

```java
// 模板
public class {{className}} {
    {{#fields}}
    private {{type}} {{name}};
    {{/fields}}
}

// 使用
className: "UserDTO"
fields: [
    {type: "String", name: "username"},
    {type: "String", name: "email"}
]

// 结果
public class UserDTO {
    private String username;
    private String email;
}
```

## 内置过滤器

```
{{name|uppercase}}      # 全大写: USER_NAME
{{name|lowercase}}      # 全小写: username
{{name|capitalize}}     # 首字母大写: Username
{{name|camelCase}}      # 驼峰: userName
{{name|pascalCase}}     # 帕斯卡: UserName
{{name|snakeCase}}      # 蛇形: user_name
```
