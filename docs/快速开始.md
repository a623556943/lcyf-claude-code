# 快速开始

## 3 步上手

### 1. 进入项目目录

```bash
cd /path/to/your/java/project
```

### 2. 克隆配置到 .claude 目录

```bash
git clone https://github.com/a623556943/lcyf-claude-code.git .claude
```

### 3. 开始使用

```bash
# 开发新功能
/lcyf-new-feature 添加用户导出功能
```

---

## 目录结构

安装后的项目结构：

```
your-project/
├── .claude/                    # lcyf-claude-code 配置目录
│   ├── agents/                 # Agent 定义
│   ├── commands/               # 自定义命令
│   ├── rules/                  # 编码规范
│   ├── skills/                 # 技能模块
│   ├── docs/knowledge/         # 知识库
│   └── CLAUDE.md               # 主配置文件
├── src/
├── pom.xml
└── .gitignore
```

## 常用命令

| 命令 | 说明 |
|------|------|
| `/lcyf-new-feature <描述>` | 完整的新功能开发流程 |
| `/lcyf-code-review` | 全面代码审查 |
| `/lcyf-learn` | 知识库管理 |

### 命令示例

```bash
# 开发新功能（自动走完整流程）
/lcyf-new-feature 添加用户导出功能

# 代码审查
/lcyf-code-review

# 查看知识库
/lcyf-learn list

# 搜索知识
/lcyf-learn search "分页查询"

# 添加知识
/lcyf-learn add --type=pattern
```
