# 快速开始

## 3 步上手

### 1. 进入项目目录

```bash
cd /path/to/your/java/project
```

### 2. 克隆配置到 .claude 目录

```bash
git clone https://github.com/a623556943/lcyf-claude-code.git .claude
```

### 3. 开始使用

```bash
# 使用命令
/lcyf-新功能 添加用户导出功能
```

---

## 目录结构

安装后的项目结构：

```
your-project/
├── .claude/                    # lcyf-claude-code 配置目录
│   ├── agents/                 # Agent 定义
│   ├── commands/               # 自定义命令
│   ├── rules/                  # 编码规范
│   ├── skills/                 # 技能模块
│   ├── templates/              # 代码模板
│   └── CLAUDE.md               # 主配置文件
├── build-parent/doc/knowledge/ # 知识库（自动创建）
├── src/
├── pom.xml
└── .gitignore
```

## 更新配置

```bash
cd .claude

```

## 常用命令

| 命令 | 说明 |
|------|------|
| `/lcyf-新功能 <描述>` | 完整的新功能开发流程 |
| `/lcyf-tdd <描述>` | 测试驱动开发 |
| `/lcyf-代码审查` | 全面代码审查 |
| `/lcyf-安全扫描` | 安全漏洞检测 |
| `/lcyf-构建修复` | 修复编译错误 |
| `/lcyf-验证` | 全面验证 |

## Git 忽略配置（可选）

如果不想提交 .claude 目录到项目仓库：

```bash
echo ".claude/" >> .gitignore
```
