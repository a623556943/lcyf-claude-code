# 快速开始

## 5分钟上手

### 1. 安装

```bash
# 克隆仓库
git clone https://github.com/a623556943/lcyf-claude-code.git
cd lcyf-claude-code

# 安装依赖
npm install
```

### 2. 复制组件到 Claude Code 配置目录

**Windows (PowerShell):**
```powershell
Copy-Item -Recurse -Force agents $env:USERPROFILE\.claude\agents
Copy-Item -Recurse -Force commands $env:USERPROFILE\.claude\commands
Copy-Item -Recurse -Force skills $env:USERPROFILE\.claude\skills
Copy-Item -Recurse -Force rules $env:USERPROFILE\.claude\rules
```

**Windows (CMD):**
```cmd
xcopy /E /I /Y agents %USERPROFILE%\.claude\agents
xcopy /E /I /Y commands %USERPROFILE%\.claude\commands
xcopy /E /I /Y skills %USERPROFILE%\.claude\skills
xcopy /E /I /Y rules %USERPROFILE%\.claude\rules
```

**macOS/Linux:**
```bash
cp -r agents ~/.claude/
cp -r commands ~/.claude/
cp -r skills ~/.claude/
cp -r rules ~/.claude/
```

### 3. 初始化项目

```bash
cd /path/to/your/java/project

# 创建知识库目录
mkdir .lcyf
```

### 4. 开始使用

在 Claude Code 中输入命令：

```
/lcyf-新功能 添加用户导出功能
```

## 典型工作流程

### 开发新功能

```
1. /lcyf-新功能 添加用户批量导入功能

   自动执行:
   ├── 需求分析 (规划专家)
   ├── 架构设计 (架构专家)
   ├── TDD开发 (测试专家 + Java开发专家)
   ├── 代码审查 (代码审查专家)
   ├── 安全扫描 (安全审查专家)
   └── 知识沉淀 (学习代理)
```

### 修复Bug

```
1. /lcyf-tdd 修复用户列表分页错误

   自动执行:
   ├── 编写失败测试
   ├── 修复代码
   ├── 确保测试通过
   └── 验证覆盖率
```

### 提交前检查

```
1. /lcyf-验证

   自动执行:
   ├── 构建检查
   ├── 测试检查
   ├── 代码规范检查
   └── 安全扫描
```

## 项目结构

```

├── .lcyf/                    # lcyf配置和知识库
│   ├── config.json          # 项目配置
│   ├── learned-patterns/    # 学习到的模式

│           ├── adapter/     # 适配层

```

## 常见问题

### Q: 命令不生效？

确保已将组件正确复制到 `~/.claude/` 目录下，检查目录结构：
```
~/.claude/
├── agents/
├── commands/
├── skills/
└── rules/
```

### Q: 如何更新？

```bash
cd lcyf-claude-code
git pull
npm install
# 然后重新复制组件到 ~/.claude/ 目录
```

### Q: 如何查看版本？

查看 `lcyf-claude-code/package.json` 中的 version 字段。
