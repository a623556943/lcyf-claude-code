# 快速参考

## 安装

```bash
cd your-java-project
git clone https://github.com/a623556943/lcyf-claude-code.git .claude
```

## 使用

```bash
/lcyf-新功能 添加用户导出功能
```

## 更新

```bash
cd .claude
git pull
```

## 常用命令

| 命令 | 说明 |
|------|------|
| `/lcyf-新功能` | 完整功能开发 |
| `/lcyf-代码审查` | 代码审查 |
| `/lcyf-安全扫描` | 安全扫描 |
| `/lcyf-tdd` | 测试驱动开发 |
