# Finance模块 业务逻辑详细文档（供应商活动）

**文档创建时间**: 2026-01-28
**文档版本**: V1.0

---

## 模块职责

负责供应商活动的配置、查询与维护，包含活动基础配置、产品加佣配置及活动快照管理。

## 核心功能

### Controller
- `FeeActivitySupplierBaseConfigController.java` - 供应商活动管理入口

### Service
- `IFeeSupplierActivityBaseConfigService.java` - 供应商活动服务接口

## 功能清单

| 功能 | 描述 |
|------|------|
| 供应商活动分页 | 供应商活动列表分页查询 |
| 活动详情（编辑） | 供应商活动详情（编辑） |
| 活动详情（详情） | 供应商活动详情（详情） |
| 新增活动 | 新增供应商活动 |
| 编辑活动 | 编辑供应商活动 |
| 删除活动 | 删除供应商活动 |

## 关键设计决策

| 决策点 | 选择 | 原因 |
|--------|------|------|
| 活动编码生成 | Snowflake 前缀编码 | 保障活动编码唯一性与可追溯性 |
| 活动快照 | 基础配置/产品配置双快照 | 支撑审核与历史追溯 |
| 结算条件校验 | Controller 前置校验 | 避免非法结算配置入库 |
